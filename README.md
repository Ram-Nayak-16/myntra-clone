# Myntra Clone - E-commerce Web Application

A premium, high-fidelity Myntra clone built with modern web technologies. This project focuses on visual excellence, responsive design, and smooth user interactions to replicate the premium shopping experience of Myntra.

## 🚀 Live Demo
[https://myntra-clone-alpha-ten.vercel.app](https://myntra-clone-alpha-ten.vercel.app)

## ✨ Features
- **Modern UI/UX**: Sleek, clean design using the Outfit font and premium color palettes.
- **Dynamic Hero Carousel**: Smooth banner transitions for highlighting top offers.
- **Interactive Categories**: High-fidelity category grid for easy navigation.
- **Responsive Design**: Fully optimized for desktops, tablets, and mobile devices.
- **Dynamic Product Rendering**: Products, deals, and categories are rendered dynamically using JavaScript.
- **Bag/Cart Functionality**: Real-time bag count updates with toast notifications.
- **Premium Animations**: Subtle hover effects, micro-interactions, and smooth transitions.

## 🛠️ Technologies Used
- **HTML5**: Semantic structure for better SEO and accessibility.
- **CSS3 (Vanilla)**: Advanced styling including Flexbox, Grid, and Custom Properties (CSS Variables).
- **JavaScript (ES6+)**: Logic for dynamic content rendering, carousel functionality, and bag management.
- **Google Fonts (Outfit)**: Modern typography for a premium feel.
- **Font Awesome**: High-quality vector icons for navigation and actions.

1. Clone this repository:
   ```bash
   git clone https://github.com/Ram-Nayak-16/myntra-clone.git
   ```
2. Navigate to the project folder:
   ```bash
   cd myntra-clone
   ```
3. Open `index.html` in your favorite browser.
